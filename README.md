# C Minigame

This project is a small **console-based minigame written in C**.  
It was created to practice fundamental programming concepts such as loops, conditions, user input handling, and simple game logic.

The game runs directly in the terminal and allows the player to interact through keyboard input.

---

# Features

- Console-based gameplay
- Interactive user input
- Simple game loop
- Score / result output
- Basic game mechanics

This project focuses on **learning and practicing core C programming concepts** while creating a small playable program.

---

# Concepts Used

The project demonstrates several fundamental C concepts:

- functions
- loops (`for`, `while`)
- conditions (`if`, `else`)
- variables and data types
- user input (`scanf`)
- console output (`printf`)

These concepts are important for understanding **low-level programming and system development**.

# How to Compile

Compile the program using **GCC**.

Linux / Mac:


gcc main.c -o minigame


Run the program:


./minigame


Windows:


minigame.exe
